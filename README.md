## installation:-
### Steps
*  fork the repository
* `git clone <repository-url>` this repository.
* `cd T-Picker-Frontend`

## install dependencies
```
npm install
```
### run the development server

Running:
-   `npm start`
-   Visit your app at [http://localhost:3000](http://localhost:3000).
