## Installation:-
### Steps
*  fork the repository
* `git clone <repository-url>` this repository.
* `cd T-Picker-Frontend`

### Install dependencies
```
npm install
```
### Run The Development Server

Running:
-   `npm start`
-   Visit your app at [http://localhost:3000](http://localhost:3000).
